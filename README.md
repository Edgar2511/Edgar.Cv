# Edgar Voskanayan — CV

Personal CV/portfolio website built with React, TypeScript, and Vite.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 8** for development and building
- **Framer Motion** for animations
- **React Icons** for iconography

## Getting Started

```bash
npm install
npm run dev
```

## Customization

All CV data is centralized in `src/data/cv-data.ts`. Edit this file to update:
- Personal info (name, title, about)
- Skills and proficiency levels
- Work experience
- Projects with links
- Education
- Languages
- Contact information

## Build

```bash
npm run build
npm run preview
```

> **Note:** Requires Node.js 20.19+ or 22.12+.
